# ROS_Tutorial
A repository for hosting simple ROS tutorials.

## ROS Setup
This repository is designed to be used in Ubuntu 20.04. 

If you want to dual boot:
[Ubuntu 20.04 Install Instructions](https://ubuntu.com/download/desktop)

Setup ROS Noetic:  [ROS Install Instructions](http://wiki.ros.org/noetic/Installation/Ubuntu)

Setup ROS Workspace: [Catkin Worspace Insctructions](http://wiki.ros.org/catkin/Tutorials/create_a_workspace)

Use your favorite git tool (I recommend [GitKraken](https://www.gitkraken.com/download)) to clone the code to the src folder of your ROS workspace

## Tutorials Available
1. Simple ROS publisher node "talker"
2. Simple ROS subscriber node "listener"
3. Custom generated message "my_msg"
